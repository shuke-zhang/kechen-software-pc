长文本示例

这是一段很长很长的文本，它将会超过两行并被截断显示省略号。这个示例展示了如何检测文本是否被截断，并通过视觉指示器显示状态。当文本被截断时，状态指示器会变成红色。

状态: 文本被截断

检测文本溢出的方法

1. 使用 scrollHeight 和 clientHeight 比较

function isTextTruncated(element) {
    return element.scrollHeight > element.clientHeight;
}

2. 使用 offsetHeight 和 scrollHeight 比较

function isTextTruncated(element) {
    return element.offsetHeight < element.scrollHeight;
}

3. 完整示例代码

// 检测所有具有 line-clamp 类的元素
document.querySelectorAll('.line-clamp-2').forEach(element => {
    const isTruncated = element.scrollHeight > element.clientHeight;
    
    if (isTruncated) {
        console.log('文本被截断:', element.textContent.substring(0, 30) + '...');
        // 添加视觉指示或其它操作
    }
});
